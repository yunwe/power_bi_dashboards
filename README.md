## Portfolio Projects
In this section I will list data analytics projects briefly describing the technology stack used to solve cases.

### 1) Logistics Operations: Diagnostic Analysis Report 
This project performs a deep-dive diagnostic analysis into delivery efficiency for a Brazilian E-Commerce | OList. 
The goal is to identify why specific orders experience extreme delays ("long-tail outliers") and to provide data-driven recommendations for operational improvement.
Successfully, identified logistic bottlenecks and FIFO performance.

**Project:** [View Project](https://github.com/yunwe/olist-logistics-efficiency)

**Tech Stack:** Python 3.13.11 (pandas, numpy, geopandas, etc..)
